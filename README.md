# 🍔 Online Food Ordering System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) Food Ordering web application where users can browse food items, add them to the cart, register/login, and place orders.

## 🚀 Features

- User Registration & Login (JWT Authentication)
- Browse Food Menu
- Food Details Page
- Add to Cart
- Checkout
- Place Orders
- Order Success Page
- Protected Routes
- MongoDB Database Integration
- REST APIs
- Responsive UI

## 🛠 Tech Stack

### Frontend
- React.js
- React Router DOM
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- dotenv

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SANNKUMARSAH/Online-Food-Ordering-System.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd "server(Node.js Backend)/backend"
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000
MONGO_URI=Your_MongoDB_Connection_String
JWT_SECRET=Your_JWT_Secret
```
